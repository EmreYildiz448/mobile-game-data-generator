# Mobile Game Analytics Dashboard Hub

*(Powered by the synthetic dataset from the Mobile Game Data Generator)*

This page contains public Looker dashboards built using the synthetic dataset generated by the Mobile Game Data Generator project.

These dashboards analyze key mobile-game performance domains:

- **Acquisition**
- **Engagement**
- **Retention**
- **Monetization**
- **A/B Testing**
- **Player Progression**
- **Ad Revenue & UA Metrics**

All dashboards visualize data created using the open-source generator available here:
GitHub Repository: [mobile-game-data-generator](https://github.com/EmreYildiz448/mobile-game-data-generator)
